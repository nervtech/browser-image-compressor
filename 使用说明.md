# 文件压缩工具 - 使用说明

## 方法一：使用HTTP服务器启动（推荐，跨平台兼容）

### 1. 构建项目
```bash
npm run build
```

### 2. 启动HTTP服务器
在终端中运行：
```bash
./start-server.sh
```

或者在Finder中双击 `start-server.sh` 文件

### 3. 在浏览器中打开
脚本会显示一个网址，通常是：
```
http://localhost:8080
```

用浏览器打开这个地址即可使用应用。

### 4. 停止服务器
在终端中按 `Ctrl+C` 即可停止服务器。

---

## 方法二：直接打开HTML文件（功能受限）

你也可以直接用浏览器打开 `dist/index.html` 文件，但某些功能（如拖放文件夹）可能无法正常工作。

---

## 打包分享给其他用户

如果你想把应用分享给其他Mac用户，只需分享以下文件：

1. `dist/` 文件夹（包含构建后的网页文件）
2. `start-server.sh` 脚本

接收方只需要：
1. 解压文件
2. 双击运行 `start-server.sh`
3. 在浏览器中打开显示的网址

---

## 系统要求

- macOS 10.14 或更高版本
- 安装了 Python3、Node.js 或 PHP（Mac通常自带Python3）
- 现代浏览器（Chrome、Safari、Firefox等）

---

## 注意事项

1. 使用HTTP服务器方式时，应用只能在本地访问（localhost），其他设备无法访问
2. 如果端口8080被占用，可以修改脚本中的端口号
3. 关闭终端窗口会停止服务器
