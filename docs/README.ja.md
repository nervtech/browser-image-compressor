# 画像圧縮ツール

ブラウザベースの画像圧縮ツール。React + TypeScript + Vite で構築。ドラッグ＆ドロップ、バッチ処理、品質/サイズ制御、多言語UIに対応。

> [English](../README.md) | [中文](README.zh-CN.md)

## 機能

- **ドラッグ＆ドロップ** — 画像やフォルダをページに直接ドロップ
- **2つの圧縮モード** — 品質制御（スライダー）または目標ファイルサイズ（KB）
- **解像度固定** — 元の画像サイズを保持するオプション
- **バッチ処理** — 4つの同時ワーカーで複数画像を圧縮
- **ライブプレビュー** — 圧縮率バッジ付きのサムネイルカード
- **出力形式** — JPEG、WebP、PNG
- **多言語** — English、中文、日本語（ブラウザから自動検出）

## クイックスタート

```bash
npm install
npm run dev
```

ブラウザで http://localhost:5173 を開きます。

## 本番用ビルド

```bash
npm run build
```

出力は `dist/` ディレクトリに生成されます。任意の静的ファイルサーバーで配信できます：

```bash
npx serve dist
```

### GitHub Pages へのデプロイ

1. リポジトリを GitHub にプッシュ
2. Settings → Pages → Source で GitHub Actions を選択、またはブランチを `main`、フォルダを `/docs` に設定
3. 手動デプロイの場合：ローカルでビルドし、`dist/` を `gh-pages` ブランチにプッシュ

## 使い方

1. ブラウザでアプリを開く
2. ツールバーで圧縮設定を構成
3. 画像やフォルダをドロップゾーンにドラッグ＆ドロップ
4. **画像を圧縮** をクリックして全ファイルを処理
5. 個別ダウンロードまたはZIPで一括ダウンロード

## プロジェクト構成

```
src/
  i18n/           — ロケールコンテキストと翻訳辞書
  utils/          — 画像処理、ZIP作成
  App.tsx         — メインアプリケーションコンポーネント
  App.css         — スタイル
  main.tsx        — エントリポイント
docs/             — 多言語ドキュメント
```

## 技術スタック
